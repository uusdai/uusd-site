Build on UUSD

Programmable stablecoin infrastructure for the AI agent economy

Overview

UUSD welcomes developers and ecosystem partners to build applications and services on its infrastructure. UUSD can be used as a stable unit of value, trading asset, and settlement asset in applications, suitable for AI agents, DeFi, payment networks, and various on-chain services.

What You Can Build

Developers can build various types of applications based on UUSD.

AI Agent Applications

AI Agent uses UUSD for automated trading, resource purchase, and service settlement

DeFi Protocols

Use UUSD as a stable trading pair or liquid asset.

Machine Payments

Build API payment, computing power rental, and data trading networks

On-chain Services

Provide a stable settlement layer for the AI service market and automation applications

Smart Contracts

UUSD smart contract adopts a Modularization architecture design, is compatible with mainstream EVM standards, and has been audited by professional security agencies.

EIP-2612

Supports off-chain signature authorization (Permit), reducing user operation steps and gas consumption

Meta Transaction

Supports meta-transactions, allowing third-party payment of gas, reducing user usage barriers

Multi-chain deployment

Deployable and circulating on mainstream EVM networks

Security Architecture

UUSD smart contracts include multiple layers of security mechanisms.

  • Authority management mechanism
  • Emergency suspension mechanism
  • Contract upgrade control
  • Risk management mechanism

Integration Guide

  1. Select target chain - UUSD supports multiple mainstream EVM networks
  2. Get contract address - Get the UUSD contract address from official documentation
  3. Integrate the SDK - Integrate wallet, payment, and settlement functions
  4. Testing and deployment - Deploy to mainnet after verification on testnet
